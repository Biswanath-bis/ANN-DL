# ⚡ Power Plant Energy Prediction using ANN

## 📌 Project Overview
This project focuses on predicting the energy output of a power plant using machine learning and deep learning techniques. An Artificial Neural Network (ANN) model is used to learn patterns from environmental factors.

---

## 📊 Dataset Information
The dataset contains **9,500+ records** with the following features:

- Temperature (AT)
- Ambient Pressure (AP)
- Vacuum (V)
- Relative Humidity (RH)

🎯 Target Variable:
- Energy Output (PE)

---

## ⚙️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib / Seaborn
- Scikit-learn
- PyTorch

---

## 🔍 Project Workflow
1. Data Collection & Understanding  
2. Data Preprocessing  
   - Handling missing values  
   - Feature scaling  
3. Exploratory Data Analysis (EDA)  
4. Model Building using ANN  
5. Model Training & Evaluation  

---

## 🧠 Model Details
- Artificial Neural Network (ANN)
- Input Layer: 4 features  
- Hidden Layers: Fully connected layers  
- Output Layer: 1 (Energy Output prediction)

---

## 📈 Performance Metrics
- RMSE: **3.5 – 4.5**
- R² Score: **0.90 – 0.95**

✔ Indicates strong predictive performance

---

## 📊 Results
The model successfully predicts power plant energy output with high accuracy and low error, showing strong correlation between environmental factors and energy production.
